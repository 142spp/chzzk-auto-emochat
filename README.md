# Chzzk Auto Emoticon

치지직 채팅창에서 이모티콘을 자동/수동으로 전송할 수 있는 크롬 확장 프로그램입니다.

## 기능
- 자동/수동 이모티콘 전송
- 랜덤 이모티콘 선택
- 반복 횟수 설정
- 도배 방지 기능
- 토스트 메시지 알림

## 사용 방법
1. 크롬 확장 프로그램으로 설치
2. 치지직 채팅창에서 이모티콘 팝업을 한 번 열어서 데이터 로드
3. 단축키 또는 팝업 메뉴를 통해 자동/수동 전송 사용

## 단축키
- 수동 전송(Default) : Alt + Q
- 자동 전송 토글(Default) : Alt + W
- 단축키는 확장프로그램 설정에서 변경 가능
